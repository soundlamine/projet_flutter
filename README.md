# Calorie App

**Calorie App** est une application Flutter pour **le suivi des calories**, **la gestion des repas**, et **les rappels via notifications locales et push**.  
Elle combine stockage local (SQLite) et cloud (Firebase) pour offrir une expérience complète.
---

## Fonctionnalités

- Ajouter, modifier et supprimer des repas avec leur apport calorique.  
- Planifier des **notifications locales** pour rappeler les repas.  
- Recevoir des **notifications push** via Firebase Cloud Messaging.  
- Authentification sécurisée avec Firebase.  
- Thèmes personnalisables : mode clair et sombre.  
- Suivi et historique des conversations ou conseils nutritionnels.  

---

## Technologies

- **Flutter** pour le développement cross-platform.  
- **SQLite** pour le stockage local des données.  
- **Firebase** pour l’authentification et les notifications push.  
- **flutter_local_notifications** pour les notifications locales.  
- **provider** pour la gestion d’état.  
- **intl** et **timezone** pour la gestion des dates et fuseaux horaires.
